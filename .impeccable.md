# 菠萝荟 · nanas.com.cn Design Context

## Design Context

### Users
社区居民（25-65岁），站在便利店门口或走回家路上，手机一拿起来就要得到答案。不是坐下来研究的产品，是"顺手一查"的工具。两类核心用户：
- **C端邻居**：想知道附近谁最便宜，查完就放手机
- **B端店主**：忙着招呼顾客，管理后台要快进快出，不能学习成本

### Brand Personality
**三个词：聪明 · 朴实 · 有温度**

- 聪明：答案比你想到还快，AI在后台帮你想好了
- 朴实：不装，不花哨，社区街坊的语气
- 有温度：不是冷冰冰的工具，是认识你的邻居帮你查的

### Aesthetic Direction

**参考：微信本体**（用户原话）
- 极简到极致：留白是设计的主角
- 信息层次靠字重+字号，不靠卡片/阴影/边框
- 颜色极度克制：一个主色，大量中性
- 每个元素存在都有理由

**反参考（绝对禁止）：**
- 橙色卡片+阴影+圆角到处复制 → 模板
- 等间距网格 → 死板
- Gradient text（background-clip: text）→ AI slop
- border-left > 1px 彩色条 → AI slop
- 玻璃拟态（glassmorphism） → AI slop

**主题：浅色**
- 背景：`oklch(98% 0.008 65)` 极浅暖白
- 主色：`oklch(58% 0.18 42)` 暖橙（=#E8611A近似）— 极少用
- 文字主：`oklch(18% 0.03 60)` 深暖黑
- 文字次：`oklch(55% 0.01 60)` 中灰
- 面：`oklch(100% 0 0)` 纯白内容区 / `oklch(95% 0.008 65)` 分隔区

**字体（Web端可用）：**
- 品牌词：ZCOOL KuaiLe 或 Ma Shan Zheng（热闹市井感，非设计师字体）
- 正文：Noto Sans SC（轻量，可读性强）
- 禁止：Inter, DM Sans, Outfit, Instrument Sans（都在黑名单）

### Design Principles

1. **字重即层次** — font-weight 900/700/400/300 的对比代替卡片+阴影
2. **橙色只出现在最重要的1个地方** — 每屏1个橙色焦点，其他全中性
3. **间距有节奏** — 同块内紧（8-12px），块间松（32-48px），用间距讲语义
4. **找答案路径最短** — 无Banner，无宫格导航，无活动推荐位
5. **空白是内容** — 宁可空着，不放无意义占位

### Technical Stack
- Cloudflare Pages 静态部署
- 纯HTML/CSS/JS，无框架
- Cloudflare Pages Functions（chat.js）
- Web端可用Google Fonts（Noto Sans SC + 品牌字体）
- oklch()颜色函数（现代浏览器全支持）
